# chaoxinsigin-java（xue习通自动签到Java版）
* 一个实用小工具，解放你的右手，让他做应该做事(手动滑稽)
## 其中有两个版本
 * Main.java是单线程版（启动一次只能监控一个课程）
 * MainThread是多线程版，可同时监控多个课程
 * 发布的jar包的为多线程版本
 * 仓库中未提供fastjson-1.2.68.jar包，改开源项目由Alibaba提供，请自行在网络上下载
 * 注意：没有GUI
